# Superstore Sales Performance Analysis

## Project Overview

This project analyzes Superstore sales data using SQL and Power BI to identify sales trends, profitability, regional performance, segment contribution, and top-performing products.

## Tools Used

- SQL
- Power BI
- DAX
- Excel

## Dashboard Preview

![Dashboard](dashboard_screenshot.png)

## Key KPIs

- Total Sales: $1.89M
- Total Profit: $236K
- Profit Margin: 12.52%
- Total Orders: 9994

## SQL Analysis

Performed analysis using SQL:

- Total Sales calculation
- Total Profit analysis
- Monthly Sales Trend
- Region-wise Sales
- Segment-wise Sales
- Category Profit Analysis
- Top 5 Products by Profit

## Power BI Dashboard Features

- Sales Trend Analysis
- Regional Performance
- Segment Contribution
- Category Profit Analysis
- Top Products Analysis
